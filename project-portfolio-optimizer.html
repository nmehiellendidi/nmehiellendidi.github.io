Financial Portfolio Optimizer

An automated Excel tool using VBA and Power Query to analyze asset risk-returns and generate optimized allocations.

Overview

Automated analysis for financial advisors to evaluate portfolios and suggest efficient frontiers based on historical returns.

Tools & Techniques

  • Advanced Excel, Power Query, VBA
  • Portfolio optimization techniques (Mean-Variance)
  • Reporting and dashboard automation

Impact

Streamlined portfolio analysis and saved hours of manual computation for financial teams.
